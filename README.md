# Arduino Proximity Detector

This is a simple proximity detector using an ultrasonic sensor and 3 LEDs to indicate the distance of an object from the sensor.

## Components
- Arduino nano
- Ultrasonic Sensor (HC-SR04)
- 3 LEDs (Red, Blue, Green)
